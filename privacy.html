Privacy Policy

Last updated: July 2026

1. Overview

This policy describes what dzbanek-bot stores. Open source: KingVojtas/dzbanek-bot. Self-hosted instances are controlled by their operators.

2. Data we store

  • Discord IDs — guild, user, and channel IDs for commands, posts, and settings.
  • Music stats — plays, skips, duration, top tracks (for /stats and /top).
  • Chat leveling — per-guild XP, level, and last award time when leveling is enabled (for /rank and /leaderboard). Message content is not stored long-term.
  • Wishlists — Steam app IDs / names for sale alerts.
  • Playlists — server “Dzbanek playlist” tracks.
  • Feed dedup — seen article/deal IDs.
  • Guild settings — optional channel IDs and feature toggles from the admin web UI.
  • Admin sessions — signed cookie after Discord OAuth (identify + guilds) for the admin dashboard.
  • Daily snapshots — anonymous aggregates (servers, plays) for the public stats chart.

3. Public stats API

The public /api/stats endpoint exposes only aggregates (server count, approximate users, total plays, etc.) — not individual user IDs or messages.

4. What we don’t collect

We do not sell personal data, store free-text search history, or require email/payment.

5. Third parties

Discord, YouTube/yt-dlp, Spotify (optional), Steam, game-deals.app, Epic, lyrics APIs — each has its own policy.
