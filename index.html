Built for real servers

Everything listed here is live in the bot today: music, deals, news, chat leveling, greetings, stats, and a web admin — not a roadmap.

🎵

Music that just works

Drop a link or search — YouTube, Spotify tracks/playlists/albums, and SoundCloud. Queue up to 100 tracks with loop, shuffle, lyrics, game OSTs, and a shared Dzbanek playlist for your whole server.

  • ✓ Idle auto-disconnect (saves bandwidth)
  • ✓ In-chat control buttons (pause, skip…)
  • ✓ /game soundtrack lookup
🎮

Deals worth posting

A daily Steam digest with live prices — only games rated Very Positive or better. Epic free games when they drop. Build a personal wishlist and get DMs when your games go on sale.

  • ✓ Review-filtered (no trash sales)
  • ✓ Steam + Epic pipelines
  • ✓ Wishlist DM sale alerts
📰

News without the noise

Configurable RSS feeds (Reuters, Ars Technica, or your own) post as clean embeds on a schedule. Deduplicated across restarts so you never get flooded — previous bot messages are cleaned up automatically.

  • ✓ Cron-based polling
  • ✓ Persistent seen-store (no reposts)
  • ✓ Optional keyword filters
🏅

Chat XP & levels

Per-server leveling with cooldowns so chat can’t be spammed for XP. Members check progress with embeds — complete with emoji progress bars and guild ranks.

  • ✓ /rank · /leaderboard
  • ✓ Level-up posts in a channel you choose
  • ✓ Toggle & reset from web Admin
👋

Welcome & goodbye

Styled join and leave embeds with avatars, member count, and optional custom messages. Turn on per server and pick channels from Admin or bot config.

  • ✓ Separate welcome / goodbye channels
  • ✓ Custom templates (user, server, …)
  • ✓ Needs Server Members Intent
⚙️

Multi-server admin

Per-guild channels for news, Steam, Epic, logs, and leveling. Slash /setup for Discord-side config, or the OAuth Admin dashboard for filters, digests, and resets.

  • ✓ Music / stats per guild
  • ✓ Audit log channel embeds
  • ✓ Live public stats API

Up and running in minutes

Three steps. No complicated setup for members.

  1. 1

    Invite the bot

    Click Add to Server, pick your guild, and grant the recommended permissions (music + embeds + manage messages for cleanup).

  2. 2

    Configure channels

    Admins run /setup news, /setup steam, or /setup epic to pick channels. Check with /setup status.

  3. 3

    Play & level up

    Join voice and run /play. Chat for XP with /rank & /leaderboard, build wishlists, and check /stats.

  • /wishlist-add Add Steam game for alerts
  • /wishlist-list List your wishlist
  • /wishlist-remove Remove by name
Plus automated daily Steam digests and Epic free-game posts in configured channels.

🏅 Leveling

  • /rank Level, XP bar & guild rank
  • /leaderboard Top 10 by XP in this server
Off by default — enable per server in Admin (cooldown + level-up channel). Chat awards XP automatically when on.

📊 Stats

  • /stats Server or user activity
  • /top plays · duration · skips · tracks

⚙️ Setup

  • /setup status Show this server’s feed settings
  • /setup news Set RSS news channel (enables news)
  • /setup steam Set Steam deals channel
  • /setup epic Set Epic free-games channel
  • /setup disable Turn off news, Steam, Epic, or all
Requires Manage Server. Music works without setup. Welcome, leveling, filters & more → Admin.

FAQ

Straight answers about privacy, permissions, and features.

Does the bot store my search history? +

No free-text search log. We track aggregated play stats, skips, wishlist game IDs, feed dedup IDs, and (if leveling is on) per-guild XP / level for chat activity. See the Privacy Policy.

How do leveling and welcome messages work? +

Leveling is off by default. Admins enable it in the Admin UI, pick a level-up channel, and members use /rank and /leaderboard. XP uses a short cooldown so spam doesn’t farm levels. Welcome / goodbye embeds are also toggled per server (channels + optional custom text). The bot needs the privileged Server Members and Message Content intents for those features.

Can I change the language? +

Command UI is English. News content follows whatever language your RSS feeds use. Multi-language bot UI isn’t built yet.

Why does it need Manage Messages? +

Only to delete the bot’s own previous Steam/Epic digest before posting a new one (channel cleanup). It is not used to moderate member messages. Full permissions guide.

What music sources are supported? +

YouTube, Spotify (tracks / playlists / albums), and SoundCloud via URL or search. Full Spotify playlist expansion needs Spotify API credentials on the host.

How do I set news or deals channels? +

Server admins (Manage Server) use /setup news, /setup steam, or /setup epic and pick a text channel. /setup status shows current settings; /setup disable turns a feed off. Music works without any setup. You can also use the Admin web UI when the API is running.

Invite vs self-host? +

Use Add to Server for the public instance, or self-host from GitHub with your own token and config. Admins configure channels with /setup or the Admin page.
