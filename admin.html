Login with Discord

Requires DISCORD_CLIENT_SECRET in the bot .env and this redirect in the Discord portal: http://127.0.0.1:3848/api/auth/callback

Setup checklist

  1. Open Discord Developer Portal → OAuth2
  2. Copy Client Secret → bot .env as DISCORD_CLIENT_SECRET=…
  3. Under Redirects, add http://127.0.0.1:3848/api/auth/callback
  4. Save .env, restart the bot, then try Login again
