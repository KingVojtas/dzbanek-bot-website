Could not reach the admin API

  • Prefer opening http://127.0.0.1:3848/admin.html while the bot is running (site + API on the same port).
  • Public site (e.g. GitHub Pages) cannot call 127.0.0.1 — browsers block it. Use the local admin URL above.
  • Later, for remote admin: host the bot with HTTPS and set PRODUCTION_API_BASE in js/config.js.

Login with Discord

Requires DISCORD_CLIENT_SECRET on the bot and an OAuth2 redirect matching …/api/auth/callback in the Discord Developer Portal.

Setup checklist

  1. Open Discord Developer Portal → OAuth2
  2. Copy Client Secret → bot .env as DISCORD_CLIENT_SECRET=…
  3. Under Redirects, add …/api/auth/callback
  4. Save .env, restart the bot, then try Login again
